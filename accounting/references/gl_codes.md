# GL Code Reference

GL (General Ledger) codes categorize expenses in the Field Museum procurement card system.

## Entertainment Codes (require supplement form)

| GL Code | Category |
|---------|----------|
| 6455 | Entertainment - Meeting with Client/Vendor/Partner |
| 6460 | Entertainment - Meeting with Donor |
| 6470 | Entertainment - Internal Staff Meeting |
| 6475 | Entertainment - Employee Gift |

Any expense using one of these codes requires a **Procurement Card Reconciliation Supplement** to be filed for that month. If alcohol was purchased, Area VP approval is also required.
